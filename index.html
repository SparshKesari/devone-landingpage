DevOneCloud Features

DevOneCloud is a web application to import and run your Terraform modules and provide self-service infrastructure for your users.

Import modules

Import your modules from source repositories (Github or Gitlab).

Variables validation

validation of Terraform variables values (mandatory variables, regex-based validation).

State management

DevOneCloud saves the state of your modules.

Notifications

Be notified when your code or when your infrastructure changes.

Step by Step DevOneCloud

Here is a simple DevOneCloud process.

  • Get Started

    Login a import your modules from Github or Gitlab

  • Set up module variables

    Input your variables default values and validations.

  • Create a stack

    Create a stack and input your variable values.

  • Run your stack

    Run your stack to execute your module and provision your infrastructure.
